GUI front-end for chigger


## Tests

Run:

```
python -m unittest discover -s tests -v
```

### Code coverage

Install coverage:

```
pip install coverage
```

Generate with:

```
coverage run -m unittest discover -s tests -v
```

Generate HTML:

```
coverage html
```
